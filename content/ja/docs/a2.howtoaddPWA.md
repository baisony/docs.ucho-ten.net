# PWAの追加方法について

## What is PWA?
PWAとは、Webサイトをアプリのように扱えるようにする技術です。  

## PWAのメリット
- Ucho-tenをスマホアプリのようにホーム画面に追加できる
- ブラウザを開く必要がない
- プッシュ通知が使える
- フルスクリーンだからデザインが綺麗

## PWAの追加方法

### iOS

| 手順                                       | スクリーンショット                                                          |
|------------------------------------------|--------------------------------------------------------------------|
| 1. SafariでUcho-tenにアクセスし<br/>「共有」ボタンをタップ | <img src='../../en/docs/img/howtoaddPWA/ios_1.png' width='300px' > |
| 2. 「ホーム画面に追加」をタップ                           | <img src='../../en/docs/img/howtoaddPWA/ios_2.png' width='300px' > |
| 3. 「追加」をタップ                        | <img src='../../en/docs/img/howtoaddPWA/ios_3.png' width='300px' > |
| 4. ホーム画面にUcho-tenが追加されます                             | <img src='../../en/docs/img/howtoaddPWA/ios_4.png' width='300px' > |
| 5. Ucho-tenをタップすると<br/>PWA版が起動します                 | <img src='../../en/docs/img/howtoaddPWA/ios_5.png' width='300px' > |
